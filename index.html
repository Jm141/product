Installation Guide

1

System Requirements

  • Windows 10/11 or macOS 10.15+
  • Node.js 18+ and npm 9+
  • MongoDB 6.0+ or PostgreSQL 14+
  • 4GB RAM minimum (8GB recommended)
  • 2GB free disk space
2

Quick Start

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Start the development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
3

Configuration

Edit the .env file to configure your database and other settings:

NODE_ENV=production
PORT=3000
MONGODB_URI=mongodb://localhost:27017/aruga_accounting
JWT_SECRET=your_jwt_secret
SMTP_HOST=smtp.example.com
SMTP_PORT=587
SMTP_USER=your_email@example.com
SMTP_PASS=your_email_password
4

Post-Installation

  • Access the application at http://localhost:3000
  • Run database migrations: npm run migrate
  • Create your first admin user
  • Set up your company information
For production use, please set up a process manager like PM2.

Installation Guide

1

System Requirements

  • Windows Server 2019+ or Linux (Ubuntu 22.04 LTS)
  • Docker and Docker Compose
  • 8GB RAM minimum (16GB recommended)
  • 50GB free disk space
  • SSL Certificate (for production)
2

Quick Start with Docker

# Create a new directory
mkdir aruga-payroll
cd aruga-payroll

# Download docker-compose.yml
wget https://downloads.aruga.com/payroll/docker-compose.yml

# Start the application
docker-compose up -d

# Initialize the application
docker-compose exec app php artisan migrate --seed

# The application will be available at https://your-domain.com
3

Configuration

Edit the docker-compose.yml file to configure your environment:

services:
  app:
    image: aruga/payroll:latest
    environment:
      - APP_ENV=production
      - APP_KEY=base64:your_app_key_here
      - DB_HOST=db
      - DB_DATABASE=payroll
      - DB_USERNAME=payroll_user
      - DB_PASSWORD=your_secure_password
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - ./data:/var/www/html/storage
    depends_on:
      - db
      - redis
4

Post-Installation

  • Access the admin panel at /admin
  • Set up your company information
  • Configure payroll settings and tax tables
  • Import employee data or add manually
For production use, please set up proper SSL certificates and backups.
